## Install Lua

### Mac
`brew install lua`

https://brewformulas.org/Lua

### Linux

`curl -R -O http://www.lua.org/ftp/lua-5.3.5.tar.gz`

`tar zxf lua-5.3.5.tar.gz`

`cd lua-5.3.5`

`make linux test`
